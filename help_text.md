# Help Text

## Security & Maintenance Commands

- **!security_audit**: Run security audits on dependencies and code
  - `!security_audit full` — Run all audits (pip-audit, bandit, safety)
  - `!security_audit pip-audit` — Check for known vulnerabilities in dependencies
  - `!security_audit bandit` — Scan code for security issues (high/critical)
  - `!security_audit safety` — Check pip packages for vulnerabilities

## Memory & Moments Commands

- **!listmoments**: Lists all moments stored in the bot.
- **!del_moment**: Allows users to delete a specific moment manually.
- **!distillation**: Automatically distills moments at a 50% retention rate.
- **!prune**: Prunes unfavourited images from storage.
